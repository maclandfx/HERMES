# 🇺🇸 美国经济软着陆监控系统

> 运行时间：每周二 09:30 自动推送  
> 数据源：公开宏观数据（BLS/BEA/FOMC/DOL/Treasury等）  
> 输出目录：`D:\Hermes\评估中心\reports\us_macro\`

---

## 核心逻辑

监控 5 大维度，综合评分 0-100：

| 维度 | 权重 | 核心指标 |
|------|------|----------|
| 🔥 通胀 | 35% | 核心CPI、核心PCE、超级核心服务 |
| 📈 增长 | 30% | GDP季环比、非农新增 |
| 🏛️ 货币政策 | 15% | 联邦基金利率、10年期美债收益率 |
| 💰 信用 | 10% | 高收益债利差 |
| 👷 劳动力 | 10% | 初请失业金、失业率 |

---

## 情景判断标准

| 评分 | 情景 | 含义 |
|------|------|------|
| ≥70 | 🟢 软着陆 | 通胀回落+增长维持+就业未崩 |
| 50-69 | 🟡 中性偏软 | 经济韧性仍在，尾部风险未出清 |
| 30-49 | 🟠 浅衰退 | 增长放缓+就业转弱 |
| <30 | 🔴 硬着陆/衰退 | 通胀反弹或GDP连续负增长 |

---

## 数据输入

**方式1（推荐）**：每次运行前更新 `latest_data.json` 中的最新值

```json
{
  "cpi_core": 0.3,      // 核心CPI月环比
  "pce_core": 0.2,      // 核心PCE月环比
  "supercore": 0.3,     // 超级核心服务通胀
  "gdp_qoq": 1.8,       // GDP季环比年化
  "unrate": 4.3,        // 失业率
  "payrolls": 85000,    // 非农新增就业
  "claims": 230000,     // 初请失业金
  "fed_rate": 3.75,     // 联邦基金利率
  "yield_10y": 4.05,    // 10年期美债收益率
  "spread_hy": 3.5,     // 高收益债利差
  "djia": 39500,        // 道琼斯指数
  "nasdaq": 17500,      // 纳斯达克指数
  "usd_index": 103.5,   // 美元指数
  "gold": 2320,         // 黄金价格
  "oil_wti": 72.5,      // WTI原油
  "copper": 8500        // 铜期货
}
```

**方式2（备选）**：脚本自动爬取部分公开页面数据（成功率不保证）
